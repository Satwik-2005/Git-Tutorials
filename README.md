# Welcome App

A simple Flask web application containerized using Docker and managed with Docker Compose.

## Features

- Flask Web Application
- Docker Support
- Docker Compose Integration
- Redis Container Support
- Easy Deployment

## Project Structure

```
DockerExample/
│
├── app.py
├── Dockerfile
├── docker-compose.yml
├── Requirements.txt
└── README.md
```

## Prerequisites

- Python 3.x
- Docker
- Docker Compose

## Build Docker Image

```bash
docker build -t web-app .
```

## Run Using Docker Compose

```bash
docker compose up
```

## Access Application

Open your browser and visit:

```text
http://localhost:5000
```

## Stop Containers

```bash
docker compose down
```

## Technologies Used

- Flask
- Docker
- Docker Compose
- Redis
